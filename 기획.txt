1. 프로젝트 기능
여러 개의 프로젝트를 만들고 관리할 수 있는 기능이다.
1-1. 프로젝트 관리 : 프로젝트를 생성, 수정, 삭제를 할 수 있어야 한다
1-2. 별도의 패널 : 프로젝트를 열면 동적으로 패널이 생성된다

2. 오버레이 기능
캔버스 위에 이미지 오버레이를 표시하는 기능.
아래의 다양한 기능이 추가로 있다
2-1. 이미지 업로드 : 이미지를 업로드할 수 있다.
업로드된 이미지의 편집, 크기 조절 등의 기능은 없다
2-2. 중심점 지정 기능 : 캔버스의 특정 지점을 클릭해서 이미지가 표시될 위치를 지정할 수 있다. 설정된 좌표는 표시된다
2-2-1. 중심점 위치 변경 : 9가지의 중심점 위치(왼쪽 상단, 가운데 상단, 오른쪽 상단, 왼쪽 가운데.... 이렇게 9개) 중에서 1개를 선택해서 이미지의 앵커 위치를 변경할 수 있다.
2-2-2. 좌표 세부 조정 : X, Y 좌표의 수치를 1씩 증감, 10씩 증감할 수 있다. 직접 숫자를 입력할 수도 있다
2-3. 불투명도 조정 : 이미지의 불투명도를 조절할 수 있다. 그리고 오버레이를 숨기거나 보일 수 있는 표시 토글이 같이 있다.
2-3. 모드 변경 : 그냥 픽셀이 대체되어 표시되는 모드 말고도, 방향 지시 모드가 있다. 방향 지시 모드에서는 캔버스의 픽셀 위에 작은 정사각형으로 나누어진 상태로 오버레이가 표시되며, 만약 인접한 정사각형과 동일한 색상이라면 서로 합쳐서 긴 경로 형태가 만들어진다.

3. 완성된 픽셀 표시
캔버스에 이미 칠해진 픽셀과 비교해서, 완료된 픽셀은 설정된 색상으로 표시해주는 기능
3-1. 표시 토글 : 이 토글을 켜고 끄는 것으로 완료된 픽셀 표시 기능을 켤지, 말지 변경할 수 있다
3-2. 색상 커스텀 : 완료된 픽셀이 어떤 색상으로 하이라이트 되게 할 지 설정할 수 있다
3-3. 투명도 조절 : 표시될 색상의 투명도를 변경할 수 있다

4. 칠해야 하는 단색 표시 기능
칠해야 하는 픽셀 중에서 설정된 색상만을 표시해주는 기능
4-1. 표시 토글 : 이 토글을 켜고 끄는 것으로 단색 표시 기능을 켤지, 말지 변경할 수 있다
4-2. 색상 선택 : 선택된 색상이 표시되고 있다. 이를 클릭하면 어떤 색상을 표시할 지 선택할 수 있는 작은 창이 열린다.
4-3. 색상 커스텀 : 해당 픽셀이 어떤 색상으로 하이라이트 되게 할 지 설정할 수 있다
4-4. 투명도 조절 : 표시될 색상의 투명도를 변경할 수 있다

5. 선택된 영역 반자동 설치 기능
직선, 사각형, 다각형 툴을 사용해서 생성된 점들이 있는 위치에 칠해야 하는 픽셀들을 가장 빠른 속도로 칠해준다.
5-1. 다양한 선택 도구들 : 직선, 사각형, 다각형의 점 생성 툴이 있으며, 아래에 자세히 설명한다
5-1-1. 직선 툴 : 직선 툴이 선택된 상태에서 캔버스 요소 위로 클릭을 하면 해당 위치를 시작점으로 해서 마우스 포인터 위치로 직선이 표시되기 시작하고, 이 직선을 따라서 점들이 생성된다. 두번째 클릭을 하면 직선은 사라지고 표시되던 위치대로 점들이 생성된다. (만약 자동으로 픽셀에 맞춰 점 개수를 변경하는 기능을 구현하지 못한 경우, 생성 중에 휠을 돌려 점의 개수를 조정할 수 있는 기능을 제작할 예정)
5-1-2. 사각형 툴 : 이 툴이 선택된 상태에서 캔버스 위를 클릭하면 클릭된 위치부터 커서까지 사각형이 표시되고, 그 안에 점들이 표시됨.

(더 있지만, 일단 여기까지 작성)

모든 패널의 위치, 값, 이미지 파일 등의 데이터는 로컬 스트로지에 실시간으로 저장되어서, 중간에 사이트가 닫히더라도 다시 접속하면 바로 복구가 가능해야 함. 단 추가된 점이나 직선을 그리던 중에 튕겼다고 다음번 접속 시에 점들이 보존되어 있거나, 직선 그리기가 시작되어 있는 등의 로컬 스트로지 저장은 필요 없다.
