# WPlace+

WPlace+는 wplace.live 사이트를 위한 향상된 사용자 경험을 제공하는 브라우저 확장 프로그램입니다.

## 🚀 주요 기능

| 기능 | 설명 | 사용법 |
|------|------|--------|
| **플로팅 모달** | 사이트 접속 시 자동으로 나타나는 모달 | 자동 생성 |
| **드래그 이동** | 모달을 자유롭게 이동 | 헤더 드래그 |
| **크기 조절** | 모달 크기를 자유롭게 조절 | 경계선/모서리 드래그 |
| **최소화/최대화** | 모달을 원형 아이콘으로 변환 | 헤더 더블클릭, 버튼 클릭 |
| **인터랙티브 UI** | 토글 스위치, 색상 피커, 프리셋 색상 | 섹션별 설정 |
| **다국어 지원** | 한국어/영어 언어 전환 | 헤더 드롭다운 |
| **위치 초기화** | 모달 위치와 크기를 기본값으로 리셋 | R키 더블클릭 |
| **상태 저장** | 모든 설정을 자동으로 저장 | 자동 저장 |

## 🎯 상세 기능

### 📍 모달 이동
- **헤더 드래그**: 모달 헤더를 드래그하여 이동
- **최소화 상태 드래그**: 원형 아이콘을 드래그하여 이동
- **위치 자동 저장**: 드래그 완료 시 위치 자동 저장

### 🔧 크기 조절
| 핸들 위치 | 기능 | 커서 |
|-----------|------|------|
| **왼쪽 경계선** | 너비 감소, 위치 오른쪽 이동 | ↔️ |
| **오른쪽 경계선** | 너비 증가 | ↔️ |
| **하단 경계선** | 높이 증가 | ↕️ |
| **오른쪽 하단 모서리** | 너비와 높이 동시 증가 | ↗️ |
| **왼쪽 하단 모서리** | 너비 감소 + 높이 증가 + 위치 이동 | ↖️ |

### 📱 최소화/최대화
| 방법 | 동작 | 상태 | 애니메이션 |
|------|------|------|------------|
| **헤더 더블클릭** | 최소화/최대화 토글 | 모든 상태 | 오른쪽 상단 기준 |
| **최소화 버튼(－)** | 최소화/최대화 토글 | 모든 상태 | 오른쪽 상단 기준 |
| **원형 아이콘 더블클릭** | 최대화 | 최소화 상태에서만 | 오른쪽 상단 기준 |
| **원형 아이콘 드래그** | 이동 | 최소화 상태에서만 | - |

### 🎨 인터랙티브 UI
- **설정 및 도구 섹션**:
  - 자동 저장 토글 스위치
  - 실시간 미리보기 토글 스위치
  - 저장 버튼 (애니메이션 효과)
  - 초기화 버튼 (확인 대화상자)
- **색상 설정 섹션**:
  - 기본 색상 피커 (하늘색 #4A90E2)
  - 보조 색상 피커 (연한 하늘색 #7BB3F0)
  - 배경 색상 피커 (매우 연한 하늘색 #E6F2FF)
  - HEX 입력 필드 (실시간 동기화)
  - 8개 프리셋 색상 버튼 (하늘색 계열)
- **섹션 관리**: 접기/펼치기 가능한 섹션
- **상태 저장**: 모든 UI 설정 자동 저장

### 🌐 다국어 지원
- **지원 언어**: 한국어, 영어
- **언어 전환**: 헤더 우측 상단 드롭다운 메뉴
- **국기 표시**: 각 언어 옵션에 해당 국가 국기 이모지 표시
- **실시간 변경**: 언어 선택 시 즉시 UI 텍스트 변경
- **설정 저장**: 선택한 언어 자동 저장 및 복원
- **드롭다운 디자인**: 프로스티드 글라스 효과와 부드러운 애니메이션

### 🔄 초기화 기능
- **R키 더블클릭**: 모달 위치와 크기를 기본값으로 초기화
- **자동 최대화**: 최소화 상태에서도 자동으로 최대화 후 초기화
- **스토리지 리셋**: 저장된 위치/크기 정보 삭제

## 🛠️ 설치 방법

4. 이 폴더 선택

## 🚨 문제 해결

### 확장 프로그램 업데이트 후 모달이 안 보이는 경우
이 문제는 다음과 같은 원인으로 발생할 수 있습니다:

1. **잘못된 위치 데이터**: 이전에 저장된 위치가 화면 밖에 있음
2. **CSS 초기 상태**: 모달이 `opacity: 0, visibility: hidden` 상태로 고정됨
3. **스토리지 데이터 손상**: 확장 프로그램 업데이트 시 저장된 데이터가 손상됨

**해결 방법:**
- **R키 더블클릭**: 모달 위치와 크기를 초기화
- **브라우저 새로고침**: 페이지를 새로고침하여 모달 재생성
- **확장 프로그램 재로드**: Chrome 확장 프로그램 관리에서 재로드

### 자동 복구 기능
- 모달 생성 시 잘못된 데이터 자동 검증 및 수정
- 화면 밖에 있는 모달 자동 감지 및 중앙 이동
- 콘솔에서 복구 과정 확인 가능

- **타이포그래피**: 세련된 산세리프 폰트
- **인터랙션**: 풍부한 호버/액티브 효과와 부드러운 애니메이션

### **테마 특징**
- **프로스티드 글라스**: 배경 블러 효과로 고급스러운 느낌
- **하늘색 톤**: 차분하고 편안한 하늘색 계열 색상
- **투명도 활용**: 다양한 투명도로 레이어링 효과
- **인터랙티브 효과**: 호버 시 스케일 변화, 그림자 강화
- **가독성 향상**: 아웃라인과 대비를 통한 명확한 구분
- **현대적 디자인**: Glassmorphism과 투명도 기반 디자인

## 🏗️ 모듈 구조

### **핵심 모듈**
- **`core.js`**: 상수, 스토리지 관리
- **`helpers.js`**: 유틸리티 함수, 데이터 검증
- **`app.js`**: 메인 애플리케이션, 초기화 및 정리

### **기능 모듈**
- **`modal.js`**: 모달 DOM 생성 및 관리
- **`drag.js`**: 드래그 이동 기능
- **`resize.js`**: 크기 조절 기능
- **`controls.js`**: UI 컨트롤 (최소화/최대화)
- **`keyboard.js`**: 키보드 이벤트 처리
- **`ui.js`**: 인터랙티브 UI 요소 관리
- **`language.js`**: 다국어 지원 및 언어 관리
- **`helpers.js`**: UI 콘텐츠 생성 및 유틸리티

### **스타일 모듈**
- **`variables.css`**: CSS 변수 정의 (하늘색 테마)
- **`modal.css`**: 모달 기본 스타일
- **`header.css`**: 헤더 스타일
- **`content.css`**: 컨텐츠 영역 스타일
- **`sections.css`**: 섹션 스타일
- **`controls.css`**: 컨트롤 요소 스타일
- **`colors.css`**: 색상 관련 스타일
- **`resize.css`**: 리사이즈 핸들 스타일
- **`responsive.css`**: 반응형 디자인

### **모듈 간 의존성**
```
app.js → 모든 모듈
modal.js → core.js, helpers.js
drag.js → modal.js, core.js
resize.js → modal.js, core.js
controls.js → modal.js, core.js
keyboard.js → controls.js
ui.js → core.js, helpers.js
language.js → core.js, helpers.js
helpers.js → core.js
```

## 🔧 개발자 도구

브라우저 콘솔에서 사용 가능한 디버깅 명령어:

```javascript
// 저장된 모든 데이터 확인
WPlacePlus.debug.showStorage()

// 모달 정보 확인
WPlacePlus.debug.showModalInfo()

// 스토리지 초기화
WPlacePlus.debug.clearStorage()

// 모달 인스턴스 확인
WPlacePlus.modal()

// 수동으로 최소화/최대화
WPlacePlus.toggleMinimize()



// 수동으로 위치 초기화
WPlacePlus.resetModalPosition()
```

- **모듈화**: 기능별 파일 분리로 유지보수성 향상
- **단일 책임 원칙**: 각 모듈이 하나의 명확한 책임을 가짐
- **프리미엄 그라데이션 디자인**: 아름답고 고급스러운 현대적 인터페이스
